# 1. 나머지 구하기

> **문제설명**
> 정수 num1과 num2가 매개변수로 주어질 때, num1을 num2로 나눈 값에 1,000을 곱한 후 정수 부분을 return 하도록 soltuion 함수를 완성해주세요.
>
> **입출력 예**
>
> | num1 | num2 | result |
> | :--: | :--: | :----: |
> |  3   |  2   |  1500  |
> |  10  |  5   |   0    |

## 💭 나의 답안 및 풀이

```js
function solution(num1, num2) {
  return (answer = num1 % num2);
}
```

# 진료 순서 정하기

> **문제 설명**  
> 외과의사 머쓱이는 응급실에 온 환자의 응급도를 기준으로 진료 순서를 정하려고 합니다.  
> 정수 배열 emergency가 매개변수로 주어질 때 응급도가 높은 순서대로 진료 순서를 정한 배열을 return하도록 solution 함수를 완성해주세요.
>
> **제한사항**
>
> - 중복된 원소는 없습니다.
> - 1 ≤ emergency의 길이 ≤ 10
> - 1 ≤ emergency의 원소 ≤ 100
>
> **입출력 예**
>
> |       emergency       |        result         |
> | :-------------------: | :-------------------: |
> |      [3, 76, 24]      |       [3, 1, 2]       |
> | [1, 2, 3, 4, 5, 6, 7] | [7, 6, 5, 4, 3, 2, 1] |
> | [30, 10, 23, 6, 100]  |    [2, 4, 3, 5, 1]    |
>
> **입출력 예 설명**
>
> - emergency가 [3, 76, 24]이므로 응급도의 크기 순서대로 번호를 매긴 [3, 1, 2]를 return합니다.
> - emergency가 [1, 2, 3, 4, 5, 6, 7]이므로 응급도의 크기 순서대로 번호를 매긴 [7, 6, 5, 4, 3, 2, 1]를 return합니다.
> - emergency가 [30, 10, 23, 6, 100]이므로 응급도의 크기 순서대로 번호를 매긴 [2, 4, 3, 5, 1]를 return합니다.

## 💭 나의 답안 및 풀이

```js
function solution(emergency) {
  let desc = emergency.slice().sort((a, b) => b - a);
  let obj = {};
  desc.forEach((v, i) => (obj[v] = i + 1));
  let result = [];
  emergency.forEach((v, i) => (result[i] = obj[v]));
  return result;
}
```

emergency라는 배열에 slice메서드를 사용하여 원본배열이 바뀌지않게하고 sort를 사용하여 오름차순으로 정렬한 배열을 desc라고 선언하였다.  
그리고 빈 객체를 obj를 생성하였다.  
오름차순으로 정렬한 배열desc를 forEach문을 사용하여 순회하며 각요소에대해 함수내의 코드블록을 실행한다.  
코드블록을 실행하면 obj에 v값을 key로 , 해당 요소의 value는 i+1을 준다.  
이때 인덱스에 +1을 한 이유는 i가 0부터 시작하여서이다.  
그리고 답을 넣을 빈배열을 result로 생성하고, 바뀌지않은 원배열 emergency에도 forEach문을 사용하여 순회한다.  
여기서 v는 요소이고 i 는 해당요소의 index이다.  
배열을 순회하면서 빈 배열의 i번째에 객체 obj의 v값을 빈배열 result에 그값을 저장하고 반환한다.
